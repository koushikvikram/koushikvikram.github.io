Rectangular Data

Rectangular data is essentially a two-dimensional matrix with
  • rows indicating records (case, example, instance, observation, pattern, sample)
  • columns indicating features (attribute, input, predictor, variable)
To apply the concepts described in the book:
Unstructured data must be processed and manipulated so that it can be represented as rectangular data.
Data in relational databases must be extracted and put into a single table.

Data Frames and Indexes

Traditional Database tables have one or more columns designated as an index - vastly improves the efficiency of certain SQL queries.

Python - Pandas library - the DataFrame object is the basic rectangular data structure.
By default, automatic integer index based on the order of rows created.
Possible to set multilevel/hierarchical indexes to improve the efficiency of certain operations.

R - the data.frame object is the basic rectangular data structure.
By default, automatic integer index based on the order of rows created.
